SRMP: Search-Based Robot Motion Planning Library

Itamar Mishani★, Yorai shaoul★, Ramkumar Natarajan★, Jiaoyang Li. Maxim Likhachev,

Carnegie Mellon University

★Equal contribution.
Paper arXiv

SRMP is a robot motion planning software that efficiently generates consistent motion plans for one or many robots. SRMP supports modern simulators, is available through C++ and Python APIs, and integrates with MoveIt! via a plugin.
